### Setup

```
npm install
react-native link
cd ios &&  pod install && cd ..
```

### Run metro bundler
```
npm start
```

### Run in iOS simulator (Mac only in new Tab)

```
npm run ios
```

### To Change react native code open App.js and Press CMD+R to refresh in simulator


### Clean metro builder (On new install)
```
npm run clean:perfect
```
